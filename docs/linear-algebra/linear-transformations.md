# Linear transformations

Linear transformations are functions mapping vectors while preserving the
properties of vector addition and scalar multiplication.

Let $V$ and $W$ be two vector spaces over the same field $F$.
A mapping $T: V \rightarrow W$ is called a linear transformation.

1. $T(u+v) = T(u) + T(v); ∀ u,v ∈ \mathbb{C}$
2. $T(αu) = αT(u); ∀ α ∈ \mathbb{R}$
